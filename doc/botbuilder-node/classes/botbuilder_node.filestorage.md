# Class: FileStorage


Middleware that implements a file based storage provider for a bot.

**Extends BotContext:**

*   context.storage - Storage provider for storing and retrieving objects.

**Usage Example**

    bot.use(new FileStorage({
         path: path.join(__dirname, 'storage')
    }));
